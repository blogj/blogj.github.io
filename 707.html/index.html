配置#

新建Linux虚拟机配置如下按照配置启动虚拟机就行，不用改什么全程图形化

详细主要配置#

  1. Machine i440fx-5.1
  2. Bios OVMF
  3. Primary vDisk Location MAnual
  4. Primary vDisk Bus SATA
  5. Network MAC 00:11:32:11:22:33
  6. Network Model virtio-net
